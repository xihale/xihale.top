Start up

首先需要知道递归法求 时间复杂度 T(n)
最朴素的情况是 T(n)=aT(nb)+r(n);(a,b>0)T(n)=aT(\frac{n}{b})+r(n);(a,b>0)T(n)=aT(bn​)+r(n);(a,b>0)(这里使用 r 而非 f 是因为我把渐进成本理解为每层递归的余子项 )

前置

e.g. T(n)=3T(n4)+Θ(n2)T(n)=3T(\frac{n}{4})+\Theta(n^2)T(n)=3T(4n​)+Θ(n2)

我们先来看最简单的情况:

这里 cn2cn^2cn2 是当前阶段的渐进成本
然后下面的三个 T(n4)T(\frac{n}{4})T(4n​) 则是转移成本

这里不难看出 将这颗树的所有节点的值加起来就是 T(n)T(n)T(n)

树高

设当前迭代到第 kkk 层
每一层都是一次迭代, 我们当然是要最终迭代到 T(1)T(1)T(1) 的
那么, 在 kkk 到最后一层时有 nbk=1⇒k=log⁡bn\frac{n}{b^k}=1 \Rightarrow k=\log_b nbkn​=1⇒k=logb​n

叶子节点数量

每次迭代, 转移的数量都是 aaa
也就是说 最后一层是 ak⇒alog⁡bna^k \Rightarrow a^{\log_b n}ak⇒alogb​n
这里我们用对数的性质可以得到: alog⁡bn=nlog⁡baa^{\log_b n}=n^{\log_b a}alogb​n=nlogb​a

证明

首先证明: alogan=na^{log_a n}=naloga​n=n(将指数拆为对数形式易得)

alog⁡bn=blog⁡b(alog⁡bn)=blog⁡bn⋅log⁡ba=blog⁡ba⋅log⁡bn=blog⁡b(nlog⁡ba)=nlog⁡ba\begin{split} a^{\log_b n}&=b^{\log_b (a^{\log_b n})}\\ &=b^{\log_b n\cdot \log_b a}\\ &=b^{\log_b a\cdot \log_b n}\\ &=b^{\log_b (n^{\log_b a})}\\ &=n^{\log_b a} \end{split} alogb​n​=blogb​(alogb​n)=blogb​n⋅logb​a=blogb​a⋅logb​n=blogb​(nlogb​a)=nlogb​a​

总成本

这个比较复杂, 我们先列出公式(从递归图得来, 最后一层+其他层)

T(n)=aT(nb)+r(n);(a,b>0)=Θ(nlog⁡ba)+∑k=0logbn−1akr(nbk)\begin{split} T(n)&=aT(\frac{n}{b})+r(n);(a,b>0)\\ &=\Theta(n^{\log_b a})+\sum_{k=0}^{log_b n-1}a^kr(\frac{n}{b^k}) \end{split} T(n)​=aT(bn​)+r(n);(a,b>0)=Θ(nlogb​a)+k=0∑logb​n−1​akr(bkn​)​

记

V(n)=nlog⁡baR(n)=∑k=0logbn−1akr(nbk)\begin{split} V(n)&=n^{\log_b a}\\ R(n)&=\sum_{k=0}^{log_b n-1}a^kr(\frac{n}{b^k}) \end{split} V(n)R(n)​=nlogb​a=k=0∑logb​n−1​akr(bkn​)​

此时 V(n)V(n)V(n) 和 R(n)R(n)R(n) 的大小共同决定了 T(n)T(n)T(n), 而 V(n)V(n)V(n) 是确定的, 比较简单, 我们主要来研究 R(n)R(n)R(n)

注意: 以下 log⁡ba−ϵ=log⁡b(a)−ϵ\log_b a-\epsilon=\log_b (a)-\epsilonlogb​a−ϵ=logb​(a)−ϵ

分类讨论:

T(n)={Θ(nlog⁡ba)r(n)=O(nlog⁡ba−ϵ),ϵ≥0Θ(r(n))r(n)=Ω(nlog⁡ba+ϵ),ϵ≥0, 0<c<1, n>N, ar(nb)≤cr(n)Θ(nlog⁡balog⁡p+1n)r(n)=Θ(nlog⁡balog⁡pn),k≥0T(n) = \begin{cases} \Theta(n^{\log_b a}) & r(n) = O(n^{\log_b a-\epsilon}),\epsilon \geq 0\\ \Theta(r(n)) & r(n) = \Omega(n^{\log_b a+\epsilon}),\epsilon\ge 0,\ 0<c<1,\ n>N,\ a r(\frac{n}{b}) \leq c r(n)\\ \Theta(n^{\log_b a}\log^{p+1} n) & r(n)=\Theta(n^{\log_b a}\log^p n),k\ge 0 \end{cases} T(n)=⎩⎨⎧​Θ(nlogb​a)Θ(r(n))Θ(nlogb​alogp+1n)​r(n)=O(nlogb​a−ϵ),ϵ≥0r(n)=Ω(nlogb​a+ϵ),ϵ≥0, 0<c<1, n>N, ar(bn​)≤cr(n)r(n)=Θ(nlogb​alogpn),k≥0​

性质一

r(n)=O(nlog⁡ba−ϵ),ϵ>0r(n) = O(n^{\log_b a-\epsilon}),\epsilon > 0r(n)=O(nlogb​a−ϵ),ϵ>0

证明

以下在非关键部分省略 ∑\sum∑ 的参数

R(n)=∑akr(nbk)≤∑ak⋅(nbk)log⁡ba−ϵ=nlog⁡ba−ϵ⋅∑(ablog⁡ba−ϵ)k=nlog⁡ba−ϵ⋅∑(abϵblog⁡ba)k=nlog⁡ba−ϵ⋅∑k=0log⁡bn−1bkϵ=nlog⁡ba−ϵ⋅nϵ−1bϵ−1=nlog⁡ba⋅1−1nϵbϵ−1R(n)=O(nlog⁡ba)\begin{split} R(n)&=\sum a^kr(\frac{n}{b^k})\\ &\le \sum a^k\cdot (\frac{n}{b^k})^{\log_b a-\epsilon}\\ &=n^{\log_b a-\epsilon}\cdot \sum (\frac{a}{b^{\log_b a-\epsilon}})^k\\ &=n^{\log_b a-\epsilon}\cdot \sum (\frac{ab^\epsilon}{b^{\log_b a}})^k\\ &=n^{\log_b a-\epsilon}\cdot \sum_{k=0}^{\log_b n-1} b^{k\epsilon}\\ &=n^{\log_b a-\epsilon}\cdot \frac{n^\epsilon-1}{b^\epsilon-1}\\ &=n^{\log_b a}\cdot \frac{1-\frac{1}{n^\epsilon}}{b^\epsilon-1}\\ R(n)&=O(n^{\log_b a}) \end{split} R(n)R(n)​=∑akr(bkn​)≤∑ak⋅(bkn​)logb​a−ϵ=nlogb​a−ϵ⋅∑(blogb​a−ϵa​)k=nlogb​a−ϵ⋅∑(blogb​aabϵ​)k=nlogb​a−ϵ⋅k=0∑logb​n−1​bkϵ=nlogb​a−ϵ⋅bϵ−1nϵ−1​=nlogb​a⋅bϵ−11−nϵ1​​=O(nlogb​a)​

性质三

r(n)=Θ(nlog⁡balog⁡pn),k≥0r(n)=\Theta(n^{\log_b a}\log^p n),k\ge 0r(n)=Θ(nlogb​alogpn),k≥0

证明

其实第三种情况和第一种是完全共通的
和性质一证明方法相同带入计算化简, 最终可以得到 R(n)=nlog⁡balog⁡knlog⁡bn−ΔR(n) = n^{\log_b a}\log^k n\log_b n-\DeltaR(n)=nlogb​alogknlogb​n−Δ, Δ\DeltaΔ 为负数, 我们直接忽略, 然后换底公式, 将换出来的底当作系数, 再乘即可得到最终结果

性质二

r(n)=Ω(nlog⁡ba+ϵ),ϵ≥0, 0<c<1, n>N, ar(nb)≤cr(n)r(n) = \Omega(n^{\log_b a+\epsilon}),\epsilon\ge 0,\ 0<c<1,\ n>N,\ a r(\frac{n}{b}) \leq c r(n)r(n)=Ω(nlogb​a+ϵ),ϵ≥0, 0<c<1, n>N, ar(bn​)≤cr(n)

证明

首先 R(n)=Ω(r(n))R(n) = \Omega(r(n))R(n)=Ω(r(n))，又因为 ar(nb)≤cr(n)a r(\dfrac{n}{b}) \leq c r(n)ar(bn​)≤cr(n)，只要 ccc 的取值是一个足够小的正数，且 nnn 的取值足够大，因此可以推导出：R(n)=O(r(n)R(n) = O(r(n)R(n)=O(r(n))。两侧夹逼可以得出，R(n)=Θ(r(n))R(n) = \Theta(r(n))R(n)=Θ(r(n))。

首先, R(n)=∑k=0log⁡bn−1akr(nbk)=r(n)+∑k=1log⁡bn−1akr(nbk)=Ω(r(n))R(n)=\sum_{k=0}^{\log_b n-1} a^kr(\frac{n}{b^k})=r(n)+\sum_{k=1}^{\log_b n-1} a^kr(\frac{n}{b^k})=\Omega(r(n))R(n)=∑k=0logb​n−1​akr(bkn​)=r(n)+∑k=1logb​n−1​akr(bkn​)=Ω(r(n))

然后, 每层转移都满足 ar(nb)≤cr(n)a r(\frac{n}{b}) \leq c r(n)ar(bn​)≤cr(n)
即 akr(nbk)≤ckr(n)a^k r(\frac{n}{b^k}) \leq c^k r(n)akr(bkn​)≤ckr(n)
放缩, 合并, 等比数列求和, 有

R(n)≤r(n)⋅∑k=0log⁡bn−1ck≤r(n)⋅∑k=0+∞ck=r(n)⋅1−c+∞1−c=r(n)⋅11−cR(n)=O(r(n))\begin{split} R(n)&\le r(n)\cdot \sum_{k=0}^{\log_b n-1} c^k\\ &\le r(n)\cdot \sum_{k=0}^{+\infty} c^k\\ &=r(n)\cdot \frac{1-c^{+\infty}}{1-c}\\ &=r(n)\cdot \frac{1}{1-c}\\ R(n)&=O(r(n)) \end{split} R(n)R(n)​≤r(n)⋅k=0∑logb​n−1​ck≤r(n)⋅k=0∑+∞​ck=r(n)⋅1−c1−c+∞​=r(n)⋅1−c1​=O(r(n))​

夹逼定理可得到 R(n)=Θ(r(n))R(n)=\Theta(r(n))R(n)=Θ(r(n))
